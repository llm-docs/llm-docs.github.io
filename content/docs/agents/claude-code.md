# Claude Code Guide

Claude Code works well for teams that want a terminal agent capable of codebase understanding, implementation planning, editing, and verification in one loop.

## Best Fit

- Medium to large repositories
- Tasks that require reasoning before edits
- Engineering workflows that benefit from a strong human approval model

## Rollout Advice

- Use repo-specific guidance files and conventions
- Keep tests and verification steps easy for the agent to run
- Review autonomy boundaries before broad adoption
- Track outcomes for speed, quality, and incident risk
