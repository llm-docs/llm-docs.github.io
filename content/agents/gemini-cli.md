# Gemini CLI

Gemini CLI is Google's open-source command-line agent for coding and local automation tasks. It combines terminal execution with built-in tools and MCP integrations.

## What It Is Good At

- Fixing bugs and building features from the terminal
- Mixing code tasks with web search or fetch workflows
- Extending the agent through MCP servers
- Running headless or scripted agent workflows

## Operating Style

Gemini CLI is a good fit when teams want a terminal coding agent that also works well as a general-purpose local automation tool.

## Notes

It is especially useful when the workflow needs model access, command execution, and external tool integration from one CLI surface.
